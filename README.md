# redvirt-metrics-failure-prediction

Скрипты для сбора метрик производительности хостов виртуализации через REST API, их анализа и обучения модели прогнозирования отказов в среде РЕД Виртуализация.

## Назначение

Проект предназначен для:
- Сбора метрик CPU, памяти и swap с узлов виртуализации через REST API
- Вычисления производных признаков (скользящие средние, скорость изменения)
- Обнаружения аномалий на основе статистических порогов
- Обучения модели Random Forest для прогнозирования отказов за 10–15 минут

## Состав репозитория

| Файл | Назначение |
|------|------------|
| `collect_metrics.py` | Сбор метрик через REST API, запись в CSV |
| `analyze_metrics.py` | Вычисление признаков, визуализация, обнаружение аномалий |
| `train_model.py` | Обучение модели Random Forest, оценка качества |
| `monitor_and_alert.py` | Мониторинг в реальном времени с отправкой алертов |
| `failure_model.joblib` | Обученная модель прогнозирования |
| `*.png` | Графики для статьи (ROC-кривая, важность признаков, матрица ошибок и др.) |
| `metrics.csv` | Пример собранных метрик (1200 записей) |
| `clean_metrics.csv` | Данные с вычисленными признаками |
